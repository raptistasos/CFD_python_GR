Εισαγωγή στη γλώσσα προγραμματισμού Python#

Περίληψη#

Σε αυτό το κεφάλαιο γίνεται μια αρχική εισαγωγή στη γλώσσα προγραμματισμού Python. Περιγράφεται η διαδικασία εγκατάστασης της γλώσσας, καθώς και οι σημαντικότερες βιβλιοθήκες της που θα χρησιμοποιηθούν στα επόμενα κεφάλαια για την επίλυση συστημάτων Μ.Δ.Ε. που περιγράφουν προβλήματα της Ρευστομηχανικής. Το κεφάλαιο κλείνει με εκτενή αναφορά στην υπολογιστική πλατφόρμα jupyter notebooks.

Προαπαιτούμενη γνώση: Βασικές γνώσεις προγραμματισμού.

Η Python είναι μια γενικού σκοπού και υψηλού επιπέδου γλώσσα προγραμματισμού με εξαιρετική αναγνωσιμότητα και ευελιξία. Χρησιμοποιείται πλέον σε ένα ευρύ φάσμα εφαρμογών, από τις πιο απλές μέχρι τις πιο περίπλοκες, στην Επιστήμη των Δεδομένων, τη Mηχανική Μάθηση, την Τεχνητή Nοημοσύνη και την Yπολογιστική Μηχανική. Η Python έχει ανέλθει τα τελευταία χρόνια στην κορυφή της λίστας δημοτικότητας σύμφωνα με το δείκτη PYPL (PopularitY of Programming Languages), έχοντας ξεπεράσει τις γλώσσες προγραμματισμού Java και C/C++.

Μια συνήθης πρακτική για την εκμάθηση ή την εφαρμογή των αρχών της Υπολογιστικής Ρευστοδυναμικής είναι η χρήση των πακέτων λογισμικού Matlab ή Mathematica. Τόσο το Matlab όσο και το Mathematica είναι, ωστόσο, εμπορικά πακέτα λογισμικού και απαιτούν συνδρομή. Στο παρόν σύγγραμμα προκρίθηκε η λύση της Python όχι μόνο επειδή είναι ελεύθερη (άνευ κόστους) και ανοιχτού κώδικα (open source), αλλά κυρίως επειδή πλαισιώνεται από ένα ευρύ οικοσύστημα επιλέξιμων βιβλιοθηκών ή πακέτων λογισμικού που επαυξάνουν σημαντικά τις δυνατότητές της. Για παράδειγμα, μια πολύ διαδεδομένη βιβλιοθήκη είναι η Numpy που προσδίδει στην Python λειτουργικότητα τύπου Matlab, υποστηρίζοντας πράξεις με μεγάλους, πολυδιάστατους πίνακες. Αντίστοιχα, η βιβλιοθήκη Sympy επιτρέπει, όπως η Mathematica, συμβολικούς μαθηματικούς υπολογισμούς. Για την κατασκευή διαγραμμάτων και, γενικότερα, στατικών ή δυναμικών απεικονίσεων διατίθενται, επίσης, πολλές βιβλιοθήκες εκ των οποίων πολύ γνωστή είναι η Matplotlib.

Στις επόμενες ενότητες θα δοθούν πρακτικές οδηγίες για την εγκατάσταση της Python και των βιβλιοθηκών της και θα παρουσιαστούν οι βασικές τους αρχές. Καθ’ όλη την έκταση του συγγράμματος, υπάρχουν ενσωματωμένα αποσπάσματα κώδικα ως εφαρμογές της θεωρίας. Ο συνδυασμός φυσικής γλώσσας, όπως τα ελληνικά, με αποσπάσματα κώδικα, ορίζεται ως «λογοτεχνικός προγραμματισμός» (literate programming) και αποτελεί, ίσως, τον πιο άμεσο και αποτελεσματικό τρόπο εκμάθησης της θεωρίας. Στην ηλεκτρονική μορφή του συγγράμματος, παρέχεται, επιπλέον, η δυνατότητα διάδρασης, δηλαδή η εκτέλεση των αποσπασμάτων κώδικα, καθώς και η κατ’ επιλογή παραλλαγή (π.χ., η αλλαγή τιμών των παραμέτρων) και η επανεκτέλεσή τους. Φυσικά, οποιαδήποτε μετατροπή κώδικα δεν επηρεάζει μόνιμα το περιεχόμενο του συγγράμματος, επομένως οι αναγνώστες προτρέπονται να εκμεταλλευτούν τις διαδραστικές δυνατότητες άφοβα.

Τα αποσπάσματα κώδικα που παρουσιάζονται στο σύγγραμμα είναι επίσης διαθέσιμα στο αποθετηρίο του συγγράμματος.

Εγκατάσταση#

Ο προτεινόμενος τρόπος εγκατάστασης της Python και των βιβλιοθηκών της είναι μέσω της σουίτας Anaconda. Το Anaconda είναι ένα πρόγραμμα διαχείρισης βιβλιοθηκών που παρέχει πρόσβαση σε μια ευρεία συλλογή (>7500 βιβλιοθήκες). Η πλήρης σουίτα Anaconda εγκαθιστά αυτόματα >250 βιβλιοθήκες, εκ των οποίων οι περισσότερες δεν είναι απαραίτητες για τις ανάγκες του μαθήματος. Προτείνεται, εναλλακτικά, η εγκατάσταση της σουίτας Miniconda που είναι μια μικρότερη διανομή της Python (προσφέρεται για όλα τα λειτουργικά συστήματα, Windows, MacOS, Linux).

H διαχείριση των βιβλιοθηκών γίνεται μέσω τερματικού, του Anaconda Prompt, όπως ονομάζεται. Ανοίγουμε το τερματικό πληκτρολογώντας «Anaconda prompt» στο πεδίο αναζήτησης του λειτουργικού μας συστήματος και επιλέγοντας την ομώνυμη εφαρμογή. Στο τερματικό μπορούμε να εκτελέσουμε διάφορες εντολές για την εγκατάσταση, την ανανέωση, την απεγκατάσταση βιβλιοθηκών, καθώς και άλλες λειτουργίες. Για παράδειγμα, η λίστα των διαθέσιμων βιβλιοθηκών που εγκαθίστανται με το Anaconda ή το Miniconda εμφανίζεται με την εντολή,

> conda list

Εάν θέλουμε να εγκαταστήσουμε μια βιβλιοθήκη που δεν αναγράφεται στη λίστα (π.χ., Sympy), γράφουμε

> conda install package_name

Ενώ αν θέλουμε να ανανεώσουμε μια βιβλιοθήκη, γράφουμε

> conda update package_name

και αν θέλουμε να απεγκαταστήσουμε μια βιβλιοθήκη, γράφουμε

> conda uninstall package_name

Στο τερματικό, θα παρατηρήσετε μπροστά από τη διεύθυνση του φακέλου ότι αναγράφεται η λέξη base εντός παρενθέσεων, η οποία υποδηλώνει το περιβάλλον που είναι ενεργοποιημένο (γνωστό ως conda environment). Εκτός από διαχειριστής βιβλιοθηκών, η σουίτα Anaconda (ή Miniconda) είναι και διαχειριστής περιβάλλοντων. Κάθε περιβάλλον είναι μια απομονωμένη συλλογή βιβλιοθηκών που δεν αλληλεπιδρά με τα υπόλοιπα περιβάλλοντα. Για την κατανόηση της λειτουργικότητας ενός περιβάλλοντος, πρέπει να γίνει σαφές ότι οι βιβλιοθήκες που χρησιμοποιούμε στο πλαίσιο της ανάπτυξης ενός προγράμματος, συντηρούνται και ανανεώνονται διαρκώς. Επομένως, αν σήμερα χρησιμοποιήσουμε μια συγκεκριμένη έκδοση των βιβλιοθηκών για να αναπτύξουμε ένα πρόγραμμα, δεν είναι εγγυημένο ότι το πρόγραμμά μας θα λειτουργεί όπως αναμένεται όταν οι βιβλιοθήκες ανανεωθούν στο μέλλον. Για παράδειγμα, μια συνάρτηση που παρέχεται από μια βιβλιοθήκη ίσως να καταργηθεί ή να τροποιηθεί σε μελλοντικές εκδόσεις της βιβλιοθήκης, με αποτέλεσμα να καθιστά μη λειτουργικό το πρόγραμμα που τη χρησιμοποιεί. Τα πλεονεκτήματα του προγραμματισμού σε απομονωμένα περιβάλλοντα είναι ότι 1) μπορούμε να κατασκευάσουμε ή να καταστρέψουμε περιβάλλοντα κατ’ επιλογή, χωρίς να δημιουργούμε πρόβλημα στη βασική διανομή της Python (δηλαδή του περιβάλλοντος base), 2) και εντός ενός περιβάλλοντος μπορούμε να εγκαταστήσουμε συγκεκριμένες εκδόσεις βιβλιοθηκών, να τις ανανεώσουμε και να τις απεγκαταστήσουμε (ακόμα και στα πλαίσια πειραματισμού). Ωστόσο, το μεγαλύτερο πλεονέκτημα είναι ότι 3) εγγυόμαστε την αναπαραγωγισιμότητα (reproducibility) του προγράμματός μας, καθώς μπορούμε να καταγράψουμε τις συμβατές εκδόσεις των βιβλιοθήκών (dependecies) και να δημιουργήσουμε ένα νέο περιβάλλον, στον ίδιο ή σε άλλο υπολογιστή (που διαθέτει το Anaconda), το οποίο να περιλαμβάνει ακριβώς τις ίδιες. Πρακτικά, αν μοιραστούμε το πρόγραμμά μας με έναν συνάδελφο, εκείνος θα μπορέσει να δημιουργήσει το κατάλληλο περιβάλλον και να εκτελέσει το πρόγραμμα με τα αναμενόμενα αποτελέσματα. Πρέπει, επίσης, να τονιστεί ότι η χρήση του Anaconda ως διανομέα της Python, εξασφαλίζει τη διαλειτουργικότητα μεταξύ των λειτουργικών συστημάτων. Δηλαδή, η λειτουργία ενός προγράμματος δεν επηρεάζεται από το λειτουργικό σύστημα (Windows, MacOS, Linux), εάν έχει εγκατασταθεί η Python μέσω του Anaconda (ή του Miniconda).

Οι εντολές για τη διαχείριση των περιβαλλόντων είναι οι εξής. Για τη δημιουργία περιβάλλοντος, χρησιμοποιούμε την εντολή

> conda create --name myenv

Πληκτρολογούμε y και πατάμε enter στην ερώτηση

> proceed ([y]/n)?

Για κατασκευή ενός περιβάλλοντος με συγκεκριμένη έκδοση της Python, γράφουμε

> conda create -n myenv python=3.6

Για την ενεργοποίηση ενός περιβάλλοντος, γράφουμε

> conda activate myenv

Με την ενεργοποίηση του περιβάλλοντος, παρατηρήστε στο τερματικό την αλλαγή του ονόματος του περιβάλλοντος εντός της παρένθεσης (αντί για base). Για να εγκαταστήσουμε, να ανεώσουμε ή να απεγκαταστήσουμε μια βιβλιοθήκη εντός του περιβάλλοντος πρέπει πρώτα να το έχουμε ενεργοποιήσει.

Για την απενεργοποίηση ενός περιβάλλοντος, γράφουμε

> conda deactivate myenv

Με την απενεργοποίηση του περιβάλλοντος, παρατηρήστε την επαναφορά του περιβάλλοντος base στο τερματικό. H λίστα των διαθέσιμων περιβαλλόντων εμφανίζεται με την εντολή,

> conda info --envs

Με αστερίσκο υποδηλώνεται το περιβάλλον που είναι ενεργοποιημένο. Οι διάφορες εντολές διαχείρισης περιβάλλοντων περιγράφονται αναλυτικά στον εξής οδηγό.

Jupyter notebooks#

Ο μη ενδεδειγμένος τρόπος εκτέλεσης ενός προγράμματος Python, είναι 1) η δημιουργία ενός αρχείου με την κατάληξη .py, (π.χ., script_name.py), 2) άνοιγμα του αρχείου σε έναν επεξεργαστή κειμένου (π.χ., Νotepad), 3) συγγραφή του κώδικα και 4) εκτελέση του προγράμματος με την εντολή python script\_name.py στο τερματικό. Η πλοήγηση στους φακέλους μέσω του τερματικού γίνεται με την εντολή cd path/to/folder. Για σχετική μετακίνηση από το φάκελο στον οποίο βρισκόμαστε, οι αντίστοιχες εντολές είναι cd .. (προηγούμενος φάκελος) και cd /subfolder (επόμενος φάκελος)

Η πλοήγηση στους φακέλους μέσω του τερματικού γίνεται με την εντολή cd path/to/folder. Για σχετική μετακίνηση από το φάκελο στον οποίο βρισκόμαστε, οι αντίστοιχες εντολές είναι cd .. (προηγούμενος φάκελος) και cd /subfolder (επόμενος φάκελος).

Ο ενδεδειγμένος τρόπος ανάπτυξης και εκτέλεσης προγραμμάτων Python (και όχι μόνο) είναι με τη χρήση των λεγόμενων Integrated Development Environments (IDEs). Τα IDEs είναι προγράμματα τα οποία διαθέτουν γραφικό περιβάλλον (GUI) και παρέχουν εύχρηστα εργαλεία για τη δημιουργία αρχείων, την πλοήγηση σε φακέλους, την αυτόματη συμπλήρωση κώδικα, τον εντοπισμό σφαλμάτων, την επισήμανση σύνταξης, κ.ά. Ένα IDE θα σας εξοικονομήσει χρόνο και θα σας βοηθήσει να γράψετε καλύτερο κώδικα. Τα κοινώς χρησιμοποιούμενα και δωρεάν Python IDEs είναι το Visual Studio Code, το Atom, το Sublime Text, το Spyder και το PyCharm Community Edition.

Στο πλαίσιο του μαθήματος, το προτεινόμενο IDE είναι το JupyterLab που βασίζεται στην τεχνολογία των jupyter notebooks τα οποία αποτελούν την προμετωπίδα του σύγχρονου λογοτεχνικού και αναπαραγωγίσιμου προγραμματισμού και γνωρίζουν μεγάλη απήχηση στην επιστημονική κοινότητα. Τα jupyter notebooks είναι δωρεάν, ανοιχτού κώδικα, διαδραστικά εργαλεία που συνδυάζουν κώδικα, υπολογιστικές εξόδους, επεξηγηματικό κείμενο και πόρους πολυμέσων σε ένα μόνο έγγραφο. Για να αντιληφθείτε τις δυνατότητες της συγκεκριμένης τεχνολογίας, το ίδιο το παρόν σύγγραμμα δημιουργήθηκε στη βάση των Jupyter notebooks.

Για την εγκατάσταση του JupyterLab σε ένα περιβάλλον conda (συμπεριλαμβανομένου του περιβάλλοντος base), γράφουμε

> conda install -c conda-forge jupyterlab

Για την εκκίνηση του JupyterLab, πλοηγούμαστε αρχικά στον επιθυμητό φάκελο και εκτελούμε την εντολή:

> jupyter lab

Ως διαδικτυακή εφαρμογή, το JupyterLab ανοίγει σε μια νέα καρτέλα του προκαθορισμένου περιηγητή διαδικτύου (browser), όπως το Google Chrome ή το Mozilla Firefox (δεν είναι απαραίτητη η σύνδεση στο διαδίκτυο). Στο αριστερό πάνελ του JupyterLab, είναι εμφανής η λίστα των αρχείων και των υποφακέλων που βρίσκονται στο φάκελο όπου εκτελέσαμε την εντολή εκκίνησης του JupyterLab. Για τη δημιουργία ενός νέου Jupyter notebook, επιλέγουμε το σύμβολο + στο επάνω αριστερά μέρος του JupyterLab και έπειτα επιλέγουμε Python 3 κάτω από την υποκεφαλίδα Notebook στο κυρίως παράθυρο της εφαρμογής. Παρατηρούμε στο αριστερό πάνελ τη δημιουργία ενός νέου αρχείου με την κατάληξη .ipynb που είναι η προκαθορισμένη κατάληξη των αρχείων Jupyter notebooks, όπως ενδεικτικά παρουσιάζεται και στο Σχήμα 1. Η λειτουργικότητά τους είναι εφάμιλλη με τα notebooks του Mathematica, καθώς τα κελιά εκτελούνται πατώντας shift + enter και επιτρέπουν από τον πειραματισμό με μικρής έκτασης κώδικες μέχρι την ανάπτυξη ολόκληρων πακέτων λογισμικού. Για να τερματίσουμε το JupyterLab, αρκεί να κλείσουμε το τερματικό από το οποίο το είχαμε εκκινήσει.

Βασικές αρχές της Python#

Βασικοί τύποι δεδομένων#

Όπως οι περισσότερες γλώσσες προγραμματισμού, η Python διαθέτει όλους τους βασικούς τύπους δεδομένων, όπως ακέραιους (integers) και αριθμούς κινητής υποδιαστολής (floats), αληθοτιμές (booleans) και συμβολοσειρές (strings).

x = 5          # ανάθεση ενός ακέραιου αριθμού στη μεταβλητή x.
print(type(x)) # Η συνάρτηση type επιστρέφει τον τύπο μιας μεταβλητής; τυπώνει "<class 'int'>" (ακέραιος)
print(x)       # τυπώνει "5"
print(x + 1)   # πρόσθεση; τυπώνει "6"
print(x - 1)   # αφαίρεση; τυπώνει "4"
print(x * 2)   # πολλαπλασιαμός; τυπώνει "10"
print(x ** 2)  # δύναμη; τυπώνει "25"
x += 1         # ισοδύναμη της παράστασης x = x + 1
print(x)       # τυπώνει "6"
x *= 2         # ισοδύναμη της παράστασης x = x * 2
print(x)       # τυπώνει "12"
y = 2.5        # ανάθεση ενός αριθμού κινητής υποδιαστολής στη μεταβλητή y.
print(type(y)) 
print(y, y + 1, y * 2, y ** 2) # τυπώνει "2.5 3.5 5.0 6.25"

Η Python διαθέτει επίσης τύπο δεδομένων για μιγαδικούς αριθμούς. Μπορείτε να αναζητήσετε περισσότερες πληροφορίες στο επίσημο εγχειρίδιο της Python.

Αληθοτιμές: Οι λογικές πράξεις στην Python γίνονται μέσω αγγλικών λέξεων και όχι με σύμβολα (&& ή ||) όπως σε άλλες γλώσσες προγραμματισμού.

t = True
f = False
print(type(t)) # τυπώνει "<class 'bool'>" (αληθοτιμή)
print(t and f) # λογική πράξη AND; τυπώνει "False"
print(t or f)  # λογική πράξη OR; τυπώνει "True"
print(not t)   # λογική πράξη NOT; τυπώνει "False"
print(t != f)  # λογική πράξη XOR; τυπώνει "True"

<class 'bool'>
False
True
False
True

Συμβολοσειρές: Η Python έχει ευρεία υποστήριξη συμβολοσειρών. Ωστόσο, δεν προσφέρει ξεχωριστό τύπο δεδομένων για μεμονωμένους χαρακτήρες όπως πολλές άλλες γλώσσες προγραμματισμού. Ένας μεμονωμένος χαρακτήρας ορίζεται στην Python απλώς ως μια συμβολοσειρά με μέγεθος 1. Οι αγκύλες μπορούν να χρησιμοποιηθούν για την πρόσβαση στους χαρακτήρες μιας συμβολοσειράς.

hello = 'hello'    # Οι συμβολοσειρές δηλώνονται με μονά εισαγωγικά
world = "world"    # ή διπλά εισαγωγικά
print(hello)       # τυπώνει "hello"
print(hello[0])    # τυπώνει τον πρώτο χαρακτήρα, δηλαδή το "h"
print(len(hello))  # η συνάρτηση len επιστρέφει το μέγεθος της συμβολοσειράς; τυπώνει "5"
hw = hello + ' ' + world  # συνένωση συμβολοσειρών; παρεμβάλλεται o χαρακτήρας του κενού μεταξύ τους (' ')
print(hw)          # τυπώνει "hello world"

hello
h
5
hello world

Σύνθετοι τύποι δεδομένων#

Εκτός από τους βασικούς τύπους δεδομένων, η Python υποστηρίζει, επίσης, σύνθετες δομές δεδομένων, όπως λίστες (lists), πλειάδες (tuples) και λεξικά (dictionaries).

Λίστες: Η πιο ευέλικτη δομή δεδομένων είναι η λίστα, που έχει εν δυνάμει μεταβλητό μέγεθος και μπορεί να περιέχει στοιχεία διαφορετικών τύπων.

xs = [4, 5, 3.5]  # δημιουργία λίστας με χρήση αγκυλών; τα στοιχεία διαχωρίζονται με κόμμα 
print(len(xs))    # η συνάρτηση len επιστρέφει τον αριθμό των στοιχείων της λίστας 
print(xs[0])      # τυπώνει το πρώτο στοιχείο. Η μέτρηση στην Python ξεκινάει από το δείκτη 0, σε αντίθεση με το Matlab.
print(xs[-1])     # οι αρνητικοί δείκτες μετράνε από το τέλος της λίστας; τυπώνει "3"
xs[0] = 'foo'     # ανάθεση της συμβολοσειράς 'foo' στην πρώτη θέση της λίστας   
print(xs)         # τυπώνει "['foo', 5, 3.5]"
xs.append('bar')  # προσθήκη νέου στοιχείου στο τέλος της λίστας
print(xs)         # τυπώνει "['foo', 5, 3.5, 'bar']"
x = xs.pop()      # αφαιρεί και επιστρέφει το τελευταίο στοιχείο
print(x, xs)      # τυπώνει "bar ['foo', 5, 3.5]"

3
4
3.5
['foo', 5, 3.5]
['foo', 5, 3.5, 'bar']
bar ['foo', 5, 3.5]

Τεμαχισμός:Εκτός από μεμονωμένα στοιχεία της λίστας, η Python επιτρέπει την πρόσβαση σε υπολίστες. Η διαδικασία είναι γνωστή ως τεμαχισμός (slicing).

Η επαναληπτική διαδικασία επί των στοιχείων μιας λίστας φέρει την ακόλουθη σύνταξη,

nums = list(range(5))     # το range είναι μια έτοιμη συνάρτηση, η οποία δημιουργεί μια λίστα ν ακεραίων αριθμών ξεκινώντας από το μηδέν 
print(nums)               # τυπώνει "[0, 1, 2, 3, 4]"
print(nums[2:4])          # υπολίστα από τη θέση 2 έως 4 χωρίς να περιλαμβάνεται το στοιχείο της θέσης 4; τυπώνει "[2, 3]"
print(nums[2:])           # υπολίστα από τη θέση 2 έως το τέλος της λίστας; τυπώνει "[2, 3, 4]"
print(nums[:2])           # υπολίστα από την αρχή της λίστας έως τη θέση 2; τυπώνει "[0, 1]"
print(nums[:-1])          # οι δείκτες μπορούν να είναι αρνητικοί; τυπώνει "[0, 1, 2, 3]"
nums[1:4] = [8, 9, 10]    # ανάθεση μια νέας υπολίστας στις θέσεις 1 έως και 3 της βασικής λίστας
print(nums)               # τυπώνει "[0, 8, 9, 10, 4]"

[0, 1, 2, 3, 4]
[2, 3]
[2, 3, 4]
[0, 1]
[0, 1, 2, 3]
[0, 8, 9, 10, 4]

animals = ['dog', 'cat', 'fish']
print("before the loop")
for animal in animals:
    print(animal)
print("after the loop")

dog

Μια σημαντική παρατήρηση είναι ότι ο κώδικας εντός της δομής επανάληψης διαχωρίζεται από τον υπόλοιπο κώδικα με εσοχές (κενά μπροστά από κάθε εντολή) και όχι με κάποια λέξη κλειδί (π.χ., end), όπως συμβαίνει σε άλλες γλώσσες προγραμματισμού. Γενικά, οι εσοχές παίζουν σημαντικό ρόλο στη σύνταξη της Python.

Ο δείκτης κάθε στοιχείου της λίστας και το ίδιο το στοιχείο προσπελάζονται κατά την επαναληπτική διαδικασία με χρήση της συνάρτησης enumerate,

animals = ['dog', 'cat', 'fish']
for idx, animal in enumerate(animals):
    print(f'#{idx}: {animal}')

#0: dog
#1: cat
#2: fish

Παραπάνω παρουσιάζεται και ένας διαφορετικός τρόπος σύνθεσης μιας συμβολοσειράς, γνωστός ως f-string.

Πολλές φορές, καλούμαστε να μετασχηματίσουμε τα στοιχεία μιας λίστας. Ένας τρόπος μετασχηματισμού είναι μέσω μιας επαναληπτικής διαδικασίας,

nums = [0, 1, 2, 3, 4, 5]
squares = []
for x in nums:
    squares.append(x ** 2)
print(squares)  # [0, 1, 4, 9, 16, 25]

[0, 1, 4, 9, 16, 25]

Ωστόσο, η Python επιτρέπει τη συμπύκνωση της παραπάνω επανάληψης σε μία μόνο σειρά κώδικα. Η διαδικασία ορίζεται ως List Comprehension.

nums = [0, 1, 2, 3, 4, 5]
squares = [x ** 2 for x in nums]
print(squares)

[0, 1, 4, 9, 16, 25]

Λεξικά: Τα λεξικά (dictionaries) χρησιμοποιούνται για την αποθήκευση ζευγών από κλειδιά και τιμές.

d = {'dog': 'furry', 'cat': 'cute'}  # δημιουργία λεξικού
print(d['cat'])   # τυπώνει την τιμή του κλειδιού "cat"
print('cat' in d) # έλεγχος αν ένα κλειδί υπάρχει στο λεξικό; τυπώνει "True"
d['fish'] = 'wet' # ορισμός ενός νέου ζεύγους κλειδιού - τιμής
print(d['fish'])  # τυπώνει "wet"

cute
True
wet

Η πρόσβαση στις εγγραφές ενός λεξικού μέσω επαναληπτικής διαδικασίας επιτυγχάνεται με την εξής σύνταξη,

d = {'men': 7, 'women': 8}
for gender in d:
    number = d[gender]
    print(f'there are {number} {gender} in the lab')

there are 7 men in the lab
there are 8 women in the lab

Εναλλακτικά, υπάρχει δυνατότητα παράλληλης χρήσης των κλειδιών και των τιμών κατά την επανάληψη μέσω της μεθόδου item,

d = {'men': 7, 'women': 8}
for gender, number in d.items():
    print(f'there are {number} {gender} in the lab')

there are 7 men in the lab
there are 8 women in the lab

Πλειάδες: Η πλειάδα (tuple) είναι μη τροποποιήσιμη συλλογή στοιχείων, σε αντίθεση με τις λίστες (lists) που είναι τροποποιήσιμες, δηλαδή επιτρέπουν προσθήκη και απομάκρυνση στοιχείων κατ’ επιλογή.

t = (2,3)
print(type(t)) # τυπώνει <class 'tuple'> (πλειάδα)
print(t)

<class 'tuple'>
(2, 3)

Συναρτήσεις: Οι συναρτήσεις στην Python ορίζονται με τη λέξη κλειδί def.

def sign(x):
    if x > 0: 
        return 'positive'
    elif x < 0:
        return 'negative'
    else:
        return 'zero'

for x in [-1, 0, 1]:
    print(sign(x))

negative
zero
positive

Στον παραπάνω κώδικα, δίνεται επίσης παράδειγμα δομής ελέγχου if/elif/else σε Python.

Βιβλιοθήκη Numpy#

Η Numpy είναι πολύ δημοφιλής βιβλιοθήκη για επιστημονικούς υπολογισμούς. Για να χρησιμοποιήσουμε οποιαδήποτε βιβλιοθήκη της Python πρέπει πρώτα να την εισάγουμε στο κώδικά μας,

import numpy as np

Στο εξής, αποκτούμε πρόσβαση στα αντικείμενα και τις συναρτήσεις της βιβλιοθήκης Numpy μέσω του ψευδώνυμου που έχουμε ορίσει, δηλαδή το np στην περίπτωσή μας (θα μπορούσε να είναι οποιoδήποτε, αλλά συνηθίζεται το συγκεκριμένο). Το κεντρικό αντικείμενο ή δομή δεδομένων της Numpy είναι ο ν-διάστατος πίνακας, γνωστός ως array. Τα μονοδιάστατα ή διδιάστατα arrays γίνονται εύκολα αντιληπτά ως πίνακες με γραμμές και στήλες. Σε ν διαστάσεις, μπορούμε να φανταστούμε ένα array ως μια περίπλοκη δομή δεδομένων που περιλαμβάνει πολλαπλά arrays διάστασης ν-1, τα οποία περιλαμβάνουν αντίστοιχα arrays διάστασης ν-2 κ.λπ.

Βιβολιοθήκη Matplotlib#

H Matpolotlib είναι η βιβλιοθήκη κατασκευής διαγραμμάτων της γλώσσας προγραμματισμού Python και της αριθμητικής της επέκτασης Numpy. Παρακάτω, εισάγεται το άρθρωμα (module) matplotlib.pyplot, που παρέχει ένα σύστημα σχεδίασης παρόμοιο με εκείνο που παρέχει το Matlab.

import matplotlib.pyplot as plt
import numpy as np

x = np.arange(0, 4 * np.pi, 0.1) # τιμές στον άξονα x
y = np.sin(x) # τιμές στον άξονα y

plt.figure(figsize=(3, 2), dpi = 100)
plt.plot(x, y)
plt.show()

Για το συνδυασμό διαγραμμάτων και την προσθήκη ετικετών στους άξονες x και y, καθώς και υπομνήματος, ακολουθείται η εξής σύνταξη,

x = np.arange(0, 3 * np.pi, 0.1) # τιμές στον άξονα x
y_sin = np.sin(x) # υπολογισμός του ημιτόνου των τιμών x
y_cos = np.cos(x) # υπολογισμός του συνημιτόνου των τιμών x

plt.figure(figsize=(3, 2), dpi = 100)
plt.plot(x, y_sin)
plt.plot(x, y_cos)
plt.xlabel('άξονας x')
plt.ylabel('άξονας y')
plt.title('Τίτλος')
plt.legend(['Ημίτονο', 'Συνημίτονο'], loc='center left', bbox_to_anchor=(1, 0.5))
plt.show()

Περισσότερες πληροφορίες για τις δυνατότητες της βιβλιοθήκης Matplotlib μπορούν να αναζητηθούν στην επίσημη ιστοσελίδα.
