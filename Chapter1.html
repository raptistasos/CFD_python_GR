Εισαγωγή στη γλώσσα προγραμματισμού Python¶

Περίληψη. Σε αυτό το κεφάλαιο γίνεται μια αρχική εισαγωγή στη γλώσσα προγραμματισμού Python. Περιγράφεται η διαδικασία εγκατάστασης της γλώσσας καθώς και οι σημαντικότερες βιβλιοθήκες της που θα χρησιμοποιηθούν στα επόμενα κεφάλαια για την επίλυση των συστημάτων μερικών διαφορικών εξισώσεων που περιγράφουν προβλήματα της Ρευστοδυναμικής. Το κεφάλαιο κλείνει με εκτενή αναφορά στα jupyter notebooks.

Παρακάτω δίνεται ένα απλό παράδειγμα παράθεσης κώδικα σε γλώσσα προγραμματισμού Python τόσο εντός του παραγώμενου PDF όσο και εντός του ηλεκτρονικού-διαδραστικού συγγράματος που είναι προσβάσιμο μέσω του διαδικτύου (HTML).

from matplotlib import rcParams, cycler
import matplotlib.pyplot as plt
import numpy as np
from myst_nb import glue
plt.ion()

# Fixing random state for reproducibility
np.random.seed(19680801)

N = 10
data = [np.logspace(0, 1, 100) + np.random.randn(100) + ii for ii in range(N)]
data = np.array(data).T
cmap = plt.cm.coolwarm
rcParams['axes.prop_cycle'] = cycler(color=cmap(np.linspace(0, 1, N)))


from matplotlib.lines import Line2D
custom_lines = [Line2D([0], [0], color=cmap(0.), lw=4),
                Line2D([0], [0], color=cmap(.5), lw=4),
                Line2D([0], [0], color=cmap(1.), lw=4)]

fig, ax = plt.subplots(figsize=(10, 5))
lines = ax.plot(data)
ax.legend(custom_lines, ['Cold', 'Medium', 'Hot']);
glue("test_fig", fig, display=False)

Στον ιστότοπο του βιβλιού, στο επάνω μέρος, υπάρχει ένα εικονίδιο σε σχήμα πυραύλου που εμφανίζει μία λίστα με τις επιλογές “Binder” και “Live Code”. H πρώτη επιλογή καλεί την υπηρεσία Binder και κατασκευάζεται κατ’ απαίτηση ένα διαδραστικό περιβάλλον ανάπτυξης κώδικα με όλες τις απαραίτητες βιβλιοθήκες (απαιτείται χρόνος της τάξης των 5-10 λεπτών για την ενεργοποίησή του), δίνοντας την δυνατότα επεξεργασίας του κώδικα που παρέχεται από το βιβλίο, ο πειραματισμός με νέες παραμέτρους αλλά και η προσθήκη νέου κώδικα. Το περιβάλλον είναι ουσιαστικά ένα jupyter notebook επομένως κάθε κελί εκτελείται πατώντας shift+enter. Αντίστοιχη είναι και η λειτουργία που παρέχει η επιλογή “Live Code” με την διαφορά ότι τα κελιά μετατρέπονται “επί τόπου” σε εκτελέσιμα (εμφανίζεται η ένδειξη “Launching from mybinder.org: ready” όταν ενεργοποιηθεί η υπηρεσία έπειτα από ορισμένο χρόνο). Σε κάθε περίπτωση, οποιαδήποτε αλλαγή από χρήστες στον κώδικα του βιβλίου έχει μόνο τοπικό χαρακτήρα και δεν επηρεάζει μόνιμα το περιεχόμενο. Περισσότερες και πιο αναλυτικές πληροφορίες σχετικά με την διαδραστικότητα του περιεχομένου και τις δυνατότητες του ηλεκτρονικού συγγράματος θα παρέχονται σε επόμενες εκδόσεις.

Στην δεξιά πλευρά του ιστότοπου του βιβλίου, υπάρχει μία πλευρική στήλη (sidebar) που δίνει την δυνατότητα εισαγωγής σημειώσεων και σχολίων μέσω της υπηρεσίας Hypothesis. Επιλέγοντας οποιοδήποτε τμήμα του βιβλίου (λέξη, φράση, πρόταση, παράγραφο κτλ.), εμφανίζονται τα εικονίδια “annotate” και “highliht”. Αν επιλεχθεί το “annotate”, η πλευρική στήλη επεκτείνεται επιτρέποντας την εισαγωγή και δημοσίευση σχολίου είτε προς μια ομάδα χρηστών είτε προς όλους δημόσια, ενώ επιλέγοντας το “highlight”, η φράση, πρόταση κτλ., χρωματίζεται με κίντρινο χρώμα. Η υπηρεσία είναι ελεύθερη, απλά προϋποθέτει τη δημιουργία λογαριασμού.

Στο τέλος κάθε σελίδας του ιστότοπου, υπάρχει ένα πλαίσιο που δίνει επίσης την δυνατότητα εισαγωγής σχολίου με την διαφορά ότι το σχόλιο δεν συνδέεται με κάποια συγκεκριμένη φράση αλλά απευθύνεται συνολικά στο περιεχόμενο της συγκεκριμένης σελίδας. Η υπηρεσία παρέχεται από την εφαρμογή “utterances” του ιστόχωρου “Github”. Κάθε σχόλιο που εισάγεται στο συγκεκριμένο πλαίσιο εμφανίζεται επίσης στη στήλη “issues” στο ηλετρονικό αποθετήριο του βιβλίου στον ιστόχωρο “Github”.

Έπειτα από την ολοκλήρωση του έργου, οι υπηρεσίες σχολιασμού ενδέχεται να διατηρηθούν στα πλαίσια της μακροπρόθεσμης υποστήριξης του βιβλίου σε συνεργασία με ηλεκτρονικούς χρήστες που θα μπορούν να προτείνουν πιθανές αλλαγές ή βελτιώσεις του περιεχομένου. Ο χρήστης του ηλεκτρονικού βιβλίου μπορεί να επισκεφθεί άμεσα το αποθετήριο μέσω του λογότυπου του “Github” στην πάνω πλευρά του βιβλίου (δίπλα από το εικονίδιο στο σχήμα πυραύλου). Μέσω του αποθετηρίου θα παρέχονται επίσης όλες οι εφαρμογές και οι ασκήσεις του βιβλίου σε μορφή jupyter notebook ωστε οι φοιτητές να μπορούν τις μεταφορτώσουν τοπικά, να πειραματιστούν και να σώσουν τις αλλαγές που κάνουν (πχ. στα πλαίσια μιας εργασίας).
