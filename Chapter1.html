Παρακάτω δίνεται ένα απλό παράδειγμα παράθεσης κώδικα σε γλώσσα προγραμματισμού Python τόσο εντός του παραγώμενου PDF όσο και εντός του ηλεκτρονικού-διαδραστικού συγγράματος που είναι προσβάσιμο μέσω του διαδικτύου (HTML).

from matplotlib import rcParams, cycler
import matplotlib.pyplot as plt
import numpy as np
from myst_nb import glue
plt.ion()

# Fixing random state for reproducibility
np.random.seed(19680801)

N = 10
data = [np.logspace(0, 1, 100) + np.random.randn(100) + ii for ii in range(N)]
data = np.array(data).T
cmap = plt.cm.coolwarm
rcParams['axes.prop_cycle'] = cycler(color=cmap(np.linspace(0, 1, N)))


from matplotlib.lines import Line2D
custom_lines = [Line2D([0], [0], color=cmap(0.), lw=4),
                Line2D([0], [0], color=cmap(.5), lw=4),
                Line2D([0], [0], color=cmap(1.), lw=4)]

fig, ax = plt.subplots(figsize=(10, 5))
lines = ax.plot(data)
ax.legend(custom_lines, ['Cold', 'Medium', 'Hot']);
glue("test_fig", fig, display=False)

Στον ιστότοπο του βιβλιού, σε σελίδες που υπάρχει κώδικας, στο επάνω μέρος είναι εμφανές ένα εικονίδιο σε σχήμα πυραύλου που εμφανίζει τις επιλογές “Binder” και “Live Code”. Επιλέγοντας “Binder”, καλείται η υπηρεσία Binder και κατασκευάζεται κατ’ απαίτηση ένα διαδραστικό περιβάλλον ανάπτυξης κώδικα με όλες τις απαραίτητες βιβλιοθήκες (απαιτείται χρόνος της τάξης των 5-10 λεπτών για την ενεργοποίησή του), δίνοντας την δυνατότα επεξεργασίας του κώδικα που παρέχεται από το βιβλίο, ο πειραματισμός με νέες παραμέτρους αλλά και η προσθήκη νέου κώδικα. Το περιβάλλον είναι ουσιαστικά ένα jupyter notebook επομένως κάθε κελί εκτελείται πατώντας shift+enter. Αντίστοιχη είναι και η λειτουργία που παρέχει η επιλογή “Live Code” με την διαφορά ότι τα κελιά μετατρέπονται “επί τόπου” σε εκτελέσιμα (εμφανίζεται η ένδειξη “Launching from mybinder.org: ready” όταν ενεργοποιηθεί η υπηρεσία έπειτα από ορισμένο χρόνο). Σε κάθε περίπτωση, οποιαδήποτε αλλαγή από χρήστες στον κώδικα του βιβλίου έχει μόνο τοπικό χαρακτήρα και δεν επηρεάζει μόνιμα το περιεχόμενο. Περισσότερες και πιο αναλυτικές πληροφορίες σχετικά με την διαδραστικότητα του περιεχομένου θα παρέχονται σε επόμενες εκδόσεις του βιβλίου.

Στην δεξιά πλευρά του ιστότοπου, υπάρχει μία πλευρική στήλη (sidebar), που δίνει την δυνατότητα εισαγωγής σημειώσεων και σχολίων μέσω της υπηρεσίας Hypothesis (η πλευρική στήλη επεκτείνεται πατώντας το βελάκι). Επιλέγονας οποιοδήποτε τμήμα του βιβλίου (λέξη, φράση, πρόταση, παράγραφο κτλ.), εμφανίζονται τα εικονίδια “annotate” και “highliht”. Επιλέγοντας “annotate” εμφανίζεται στην πλευρική στήλη η δυνατότητα εισαγωγής και δημοσίευσης σχολίου είτε προς μια ομάδα χρηστών είτε προς όλους δημόσια. Η υπηρεσία είναι ελεύθερη, απλά προϋποθέτει τη δημιουργία λογαριασμού. Οι δυνατότητες εισαγωγής σημειώσεων και σχολίων θα χρησιμοποιηθούν τόσο από τους συγγραφείς κατά την ανάπτυξη του βιλίου όσο και για την άμεση συνεργασία με την κεντρική ομάδα υποστήριξης (ΚΟΥ) του έργου. Έπειτα από την ολοκλήρωση του έργου, η υπηρεσία ενδέχεται να διατηρηθεί στα πλαίσια της γενικότερης υποστήριξης του βιβλίου σε συνεργασία με ηλεκτρονικούς χρήστες που θα μπορούν να προτείνουν πιθανές αλλαγές ή βελτιώσεις του περιεχομένου.

Στο ηλετρονικό αποθετήριο του βιβλίου στον ιστόχωρο “Github” (στην πάνω πλευρά του ιστότοπου του βιβλίου υπάρχει αντίστοιχο εικονίδιο που επίσης οδηγεί στο αποθετήριο), οι χρήστες αλλά και όσοι εμπλέκονται στην ανάπτυξη του έργου (πχ. ΚΟΥ) θα μπορούν να εγείρουν ζήτημα (issue) το οποίο θα λαμβάνεται υπόψιν από τους συγγραφείς για την βελτίωση του περιεχομένου. Μέσω του αποθετηρίου θα παρέχονται επίσης όλες οι εφαρμογές και οι ασκήσεις του βιβλίου σε μορφή jupyter notebook ωστε οι φοιτητές να μπορούν τις μεταφορτώσουν τοπικά, να πειραματιστούν και να σώσουν τις αλλαγές που κάνουν (πχ. στα πλαίσια μιας εργασίας).

Επίσης πρέπει να τονιστεί και η δυνατότητα εισαγωγής πολλαπλών υπερ-συνδέσμων τόσο στο PDF αλλά και στο HTML του συγγράματος (έχει γίνει ήδη εμφανής) που οδηγούν είτε σε εξωτερικές πηγές ή παραπέμπουν σε τμήματα του βιβλίου, εξισώσεις, εικόνες, διαγράμματα, βιβλιογραφικές αναφορές, κτλ.
